--------------------
|エディタの機能説明|
--------------------


・OPENボタン
ステージ情報を記録したjsonファイルを指定して開く


・SAVEボタン
ファイル名を決め、ステージ情報をそのファイルにjson形式で保存する


・レイヤー変更ボタン
1. レイヤー0
2. レイヤー1
3. レイヤー2
4. 属性
5. アイテム
6. キャラ(今のところ敵のみ)

の順番で編集中のレイヤーを変更することができる


・キャラレイヤーを編集していないときにマス上で右クリックをするとそこがプレイヤーの初期位置になる



------------------
|各レイヤーの説明|
------------------
・レイヤー0,1,2
右枠に表示されているグラフィックを左クリックで選択し、
マス上で左クリックすることでそのマスに配置できる。

実際のゲームプレイでは0,1,2の順レイヤーが重ねて表示されることになる。

なお、右枠の一番左上の空枠を左クリックし、
その後配置したグラフィックを左クリックすることで配置操作を取り消すことができる。


・属性
〇：通れる
×：通れない
△：通れない＋動かせる
という風に各マスに属性を設定できる。
これはマス上で左クリックすることで、そのマスの属性を上記の順に変えることができる。


・アイテム
レイヤー0,1,2と操作は同じ。
レイヤー0,1,2に重ねて表示される。


・キャラ（敵）
右枠に表示されたパンダを左クリックし、
マス上で左クリックすることでそのマスに配置できる。
配置後は違う場所をクリックすることで、
巡回経路を設定することができる。
経路設定は右クリックもしくはレイヤーを変えることで終えることができる。

巡回は
初期位置->最後の位置->初期位置...
というように行ったり来たりする仕様であることに注意。

右枠の空の枠を左クリックしたのち、
配置したパンダのマスを左クリックすることで消すことができる。
